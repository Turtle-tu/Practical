# Отчёты в бинарных форматах

В данной папке размещены итоговые отчёты по проектной практике в бинарных форматах.

---

# Файлы отчётов

| Файл | Описание |
|---|---|
| report.docx | Итоговый отчёт по проектной практике в формате Microsoft Word (DOCX). Содержит титульный лист, введение, описание разработанной системы измерения объёма грузов, результаты работы, сведения о взаимодействии с организацией-партнёром, выводы и приложения. |
| report.pdf | PDF-версия итогового отчёта. Полностью соответствует DOCX-файлу и предназначена для загрузки в СДО (LMS), печати и архивного хранения. |
| report_2.docx | Итоговый отчёт по проделанной работе над сайтом и телеграм ботом. Содержит титульный лист, введение, описание разработанной системы измерения объёма грузов, результаты работы, сведения о взаимодействии с организацией-партнёром, выводы и приложения. |
| report_2.pdf | PDF-версия итогового отчёта. Полностью соответствует DOCX-файлу и предназначена для загрузки в СДО (LMS), печати и архивного хранения. |

# Дополнительные документы

Помимо бинарных отчётов, в репозитории также размещены дополнительные материалы в формате Markdown:

- individual_plans.md — индивидуальные планы работы участников проекта;
- partner_report.md — отчёт о взаимодействии с организацией-партнёром;
- technical_guide.md — техническое руководство по разработке и запуску системы;
- README.md — описание проекта, инструкции по запуску и обзор функционала системы.
