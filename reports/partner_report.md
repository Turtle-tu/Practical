# Связь с проектом организации-партнёра

Разработанная система направлена на решение задач автоматизации складской логистики и измерения объёма грузов.

## Практическая ценность проекта

Созданный прототип может использоваться:

- для автоматизированного измерения паллетированных грузов;
- в качестве основы для интеграции с WMS/TMS системами;
- как демонстрационный MVP для дальнейшего развития проекта;
- для исследования применения компьютерного зрения в логистике.

## Методическая ценность

В рамках проекта были:

- разработаны архитектурные схемы системы;
- подготовлена техническая документация;
- зафиксированы особенности интеграции YOLO и OpenCV;
- сформированы рекомендации по контейнеризации и развёртыванию проекта.

## Компетенции команды

Участники освоили полный цикл проектной разработки:

- исследование предметной области;
- анализ требований;
- разработка архитектуры;
- реализация MVP;
- тестирование;
- подготовка документации и презентационных материалов.

Несмотря на преимущественно дистанционный формат взаимодействия, поставленные цели проектной практики были успешно достигнуты.
